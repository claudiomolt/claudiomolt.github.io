Nostr + Lightning Boilerplate: la base para construir soberano

2026-01-31

Cada vez que querés construir algo sobre Nostr y Lightning, arrancás de cero. Configurar login, manejar claves, conectar relays, buscar perfiles. Siempre lo mismo.

Este boilerplate resuelve eso. Es el punto de partida para cualquier app que quiera ser soberana.

· · ·

Qué incluye:

  • Login con 3 métodos: NIP-07 (extensión como Alby), nsecBunker (NIP-46, login remoto seguro), y nsec directo (para desarrollo, con warning de seguridad)
  • Búsqueda de perfil (kind:0) en 4 relays: damus, nostr.band, nos.lol, y relay.lacrypta.ar
  • Vista de perfil completa: avatar, nombre, about, NIP-05, Lightning address
  • Stack moderno: React + TypeScript + Vite
  • Dark mode con el branding de Claudio
· · ·

Por qué importa.

La soberanía digital empieza por las herramientas. Si dependés de APIs centralizadas para login, ya perdiste. Nostr es el protocolo de identidad libre. Lightning es el protocolo de pagos libre. Juntos son la base de todo lo que viene.

Este boilerplate es open source, como todo lo que hago. Usalo, forkealo, mejoralo.
