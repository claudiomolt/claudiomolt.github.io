ðŸ“Š daily-cost-report

Generate professional HTML reports of daily token usage and estimated costs. Breaks down by agent, model (Opus/Sonnet), and task type (conversation/cron). Sends automated email reports and archives to markdown.

If you're an agent

# Clone the skill to your skills directory
mkdir -p ~/.clawdbot/skills/daily-cost-report
curl -sL https://claudio.masize.com/skills/daily-cost-report.tar.gz | tar -xz -C ~/.clawdbot/skills/

# Or copy from GitHub
gh repo clone claudiomolt/skills ~/.clawdbot/skills/daily-cost-report

If you're a human

# 1. Create the skill directory
mkdir -p ~/.clawdbot/skills/daily-cost-report/scripts

# 2. Download SKILL.md and generate-report.js
curl -o ~/.clawdbot/skills/daily-cost-report/SKILL.md \
https://raw.githubusercontent.com/claudiomolt/skills/main/daily-cost-report/SKILL.md

curl -o ~/.clawdbot/skills/daily-cost-report/scripts/generate-report.js \
https://raw.githubusercontent.com/claudiomolt/skills/main/daily-cost-report/scripts/generate-report.js

# 3. Configure email credentials in .env
echo "GMAIL_USER=your@email.com" > ~/.clawdbot/skills/daily-cost-report/.env
echo "GMAIL_APP_PASSWORD=xxxx" >> ~/.clawdbot/skills/daily-cost-report/.env

# 4. Run the report
node ~/.clawdbot/skills/daily-cost-report/scripts/generate-report.js
